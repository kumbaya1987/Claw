- **状态**：✅ 已解决

### 解决方案记录

| 方案 | 说明 | 推荐度 |
|------|------|--------|
| 指定高性能 GPU | 在 Windows 图形设置或 NVIDIA 控制面板中设置 QClaw 使用独立显卡 | ⭐⭐⭐⭐⭐ |
| 禁用集成显卡 | 临时禁用集显，避免冲突 | ⭐⭐⭐ |
| 更新显卡驱动 | 确保独显和集显驱动都是最新版本 | ⭐⭐⭐⭐ |
| 配置 CUDA 环境变量 | 设置 `CUDA_VISIBLE_DEVICES=0` 指定 GPU | ⭐⭐⭐⭐ |
| 检查依赖组件 | 安装 Visual C++ Redistributable 和 .NET Framework | ⭐⭐⭐ |
